# E-commerce tester

### Descrição:
Esse projeto foi desenvolvido para facilitar o teste visual de todas as páginas de um e-commerce, funcionando para usuários técnicos e não técnicos.

### Tecnologias:
As tecnologias utilizadas foram: Python e Selenium.
